# HTML FILE GENERATOR
Simple HTML File Generator for _**Command Prompt**_

| Option | Description |
| ------ | ----------: |
| -h --help | Prints Help |
| -t --type [INT] | Specify your wanted preset|
| *1  | Full |
| *2  | Medium |
| *3  | Minimalistic |
| *99 | Exit |

## Supported OS's
* Microsoft Windows (Coming Soon)
* Linux
